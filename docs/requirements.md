# ADB Desktop - 要件定義書

## 概要

Android Debug Bridge (adb) の頻出操作をGUIで手軽に実行できるmacOSネイティブアプリケーション。
コマンドの記憶・入力の手間を省き、ワンクリックでadb操作を完了させることを目的とする。

## 基本情報

| 項目 | 内容 |
|------|------|
| アプリ名 | ADB Desktop |
| 対象OS | macOS |
| 技術スタック | Swift / SwiftUI |
| 対象ユーザー | Android開発者（自分・チームメンバー） |
| 前提条件 | Android SDK導入済み（環境変数からadbパスを自動検出） |
| データ永続化 | UserDefaults |
| UIタイプ | 通常のウィンドウアプリケーション |
| ダークモード | 対応（SwiftUIの自動対応） |

## 機能詳細

### 1. デバイス一覧表示・接続管理

- `adb devices` の結果をリスト表示する
- デバイス名、シリアル番号、接続状態（device / offline / unauthorized）を表示する
- 接続デバイスの自動検出（ポーリングまたは手動リフレッシュ）
- v1.0では単一デバイス選択のみ対応

### 2. テキスト入力

- テキストフィールドに入力した文字列を `adb shell input text "..."` で端末に送信する
- ASCII文字のみ対応（日本語入力は対象外）
- 入力テキストの履歴を複数保存できる
  - リストから選択してワンクリックで再送信可能
  - 履歴の追加・削除が可能
- 用途例: ログインID/パスワードの入力を省力化

### 3. URLスキーム送信

- URLスキームを入力し、`adb shell am start -a android.intent.action.VIEW -d "<URL>"` で端末に送信する
- スキームをラベル付きリストで複数保存できる
  - 追加・編集・削除が可能
- テンプレート機能
  - `myapp://product/{id}` のようにプレースホルダーを含むURLを保存できる
  - 実行時にプレースホルダー部分のみ入力して送信する

### 4. スクリーンショット撮影

- ワンクリックで端末のスクリーンショットを撮影する
- 実行コマンド: `adb shell screencap` → `adb pull`
- 保存先: `~/Downloads/` 固定
- ファイル名: タイムスタンプベース（例: `screenshot_2026-03-13_143000.png`）

### 5. 画面録画

- 開始/停止ボタンで端末の画面録画を制御する
- 実行コマンド: `adb shell screenrecord`
- 録画設定: デフォルト値（解像度・時間制限はadbのデフォルトに従う）
- 保存先: `~/Downloads/` 固定
- ファイル名: タイムスタンプベース（例: `recording_2026-03-13_143000.mp4`）

### 6. Wi-Fi adb接続

- USB接続中の端末をWi-Fi接続に切り替える
  - `adb tcpip 5555` → `adb connect <device_ip>:5555`
- 接続/切断のトグル操作

### 7. プロキシ設定

- 端末のWi-Fiプロキシ設定をオン/オフ切り替える
- オン時の設定値:
  - ホスト: Macの現在のIPアドレス（`ipconfig getifaddr en0` で取得）
  - ポート: `8080`（Charles Proxy想定）
- 実行コマンド:
  - オン: `adb shell settings put global http_proxy <ip>:8080`
  - オフ: `adb shell settings put global http_proxy :0`
- 現在のプロキシ状態を表示する

### 8. コマンドプリセット実行

以下のプリセットコマンドをデフォルトで用意し、ワンクリックで実行できるようにする。

| プリセット名 | 説明 | 備考 |
|-------------|------|------|
| アプリ起動 | `adb shell am start -n <package>/<activity>` | パッケージ名・アクティビティ名を保存可能（追加・編集・削除） |
| テキスト入力 | 機能#2へのショートカット | - |
| スクリーンショット撮影 | 機能#4へのショートカット | - |
| 動画撮影 | 機能#5へのショートカット | - |
| URLスキーム実行 | 機能#3へのショートカット | - |
| Wi-Fiプロキシ設定 | 機能#7へのショートカット | - |

※ ユーザーによるカスタムプリセットの追加・編集・削除も可能とする

## adbパス解決

以下の優先順で自動検出する:

1. 環境変数 `ANDROID_HOME` → `$ANDROID_HOME/platform-tools/adb`
2. 環境変数 `ANDROID_SDK_ROOT` → `$ANDROID_SDK_ROOT/platform-tools/adb`
3. 検出できない場合はエラーメッセージを表示し、手動設定を促す

## 非機能要件

| 項目 | 内容 |
|------|------|
| 対応macOSバージョン | macOS 14 (Sonoma) 以上 |
| 配布方法 | 直接配布（.app） ※App Store配布は対象外 |
| データ保存 | UserDefaults（エクスポート/インポートは対象外） |
| エラーハンドリング | adbコマンド失敗時にアラートでエラー内容を表示 |
| パフォーマンス | adbコマンドは非同期実行し、UIをブロックしない |

---

## 対象外（v1.0）

- アプリのインストール/アンインストール
- ファイル転送（push/pull）
- 開発者オプション設定変更
- 日本語テキスト入力
- 複数デバイス同時操作
- logcat表示
- データのエクスポート/インポート
